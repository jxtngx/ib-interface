# Skill: Code Review

Proficiency in reviewing code for quality and architectural compliance.

## Competencies

- Ability to assess asyncio correctness
- Knowledge of backwards compatibility concerns
- Understanding of API design principles
- Familiarity with Python code quality standards
- Experience with identifying breaking changes

## Context

Code review ensures changes align with ib-interface's asyncio-first architecture, maintain backwards compatibility, and follow established patterns.

## Review Checklist

- asyncio patterns correct
- EventKit events emit properly
- No breaking public API changes
- Type hints present
- Tests included
