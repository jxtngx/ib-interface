# Skill: Dependency Management

Proficiency in managing Python project dependencies.

## Competencies

- Understanding of pyproject.toml configuration
- Knowledge of version specifiers and constraints
- Familiarity with optional/dev dependencies
- Understanding of dependency resolution
- Experience with uv/pip package management

## Context

ib-interface uses pyproject.toml with uv_build. Dependencies must be carefully managed to avoid conflicts and ensure reproducibility.

## Key Concerns

- Protobuf version compatibility
- Minimal dependency footprint
- Dev vs runtime dependencies
