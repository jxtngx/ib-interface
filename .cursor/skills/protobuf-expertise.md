# Skill: Protocol Buffers Expertise

Proficiency in Google Protocol Buffers for API communication.

## Competencies

- Understanding of protobuf message definition syntax (.proto files)
- Knowledge of protobuf serialization/deserialization
- Familiarity with generated Python code (*_pb2.py files)
- Understanding of optional fields and `HasField()` semantics
- Knowledge of protobuf type mappings (int32, string, bytes, etc.)

## Context

TWS API versions 10.35+ use protobuf for new message types. This skill enables working with the 204 protobuf message definitions used for orders, market data, account info, and configuration.
