# Skill: SigNoz Observability

Proficiency in SigNoz for application monitoring and log visualization.

## Competencies

- Understanding of SigNoz architecture (ClickHouse, Query Service, Frontend)
- Knowledge of OTLP ingestion endpoints
- Familiarity with SigNoz query builder and dashboards
- Understanding of log aggregation and filtering
- Experience with alerts and notification channels
- Knowledge of trace-to-log correlation

## Context

SigNoz is the primary observability backend for ib-interface, providing log visualization, trace analysis, and metrics dashboards.

## Key Concepts

- OTLP endpoint typically at port 4317 (gRPC) or 4318 (HTTP)
- Logs indexed by service name, severity, attributes
- Dashboards for connection health, order latency, error rates
- Alerts for connection drops, error spikes
