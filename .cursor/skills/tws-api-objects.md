# Skill: TWS API Objects

Knowledge of TWS API data structures and their attributes.

## Competencies

- Understanding of Contract and ContractDetails structures
- Knowledge of Order and OrderState attributes
- Familiarity with Execution and CommissionReport
- Understanding of BarData for historical/realtime data
- Knowledge of Ticker for streaming market data
- Familiarity with account and position objects

## Context

TWS API has rich data structures with many attributes. New server versions add attributes for regulatory compliance, extended hours, and advanced order types.

## Key Objects

- Contract: Security identification
- Order: Order parameters (60+ attributes)
- ContractDetails: Full contract metadata including fund data
- BarData: OHLCV bar data
